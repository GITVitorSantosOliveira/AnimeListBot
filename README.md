# AnimeListBot
A telegram bot that consists of making apis requests to Anilist

## Build With
- Javascript 
- nodejs
- [GraphQL](https://graphql.org) <img src="https://github.com/devicons/devicon/blob/master/icons/graphql/graphql-plain.svg" alt="GraphQl" width="17" height="17"/>
- [Telegram Api](https://core.telegram.org/bots/api)
- [Anilist Api](https://anilist.gitbook.io/anilist-apiv2-docs/)

The application is running on [Heroku](https://www.heroku.com) and on telegram the bot is called @listsofanimebot
